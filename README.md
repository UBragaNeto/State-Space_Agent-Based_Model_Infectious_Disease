# A-State-Space-Agent-Based-Model-for-Infectious-Disease-Spread
An Open Source repository for the codes used to generate results for the paper titled "A State-Space Agent-Based Model for Infectious Disease Spread" submitted to Computation on March 21, 2026.

These codes were originally run as Google Colab notebooks in an L4 GPU environment with 24 GB of RAM. If memory issues are encountered, uncomment the memory allocation line 8.
